# CyberGuard

CyberGuard is a Flask-based cybersecurity monitoring dashboard designed to track and manage security alerts.

## Features

* Security Alert Dashboard
* SQLite Database Integration
* Add Security Alerts
* Delete Security Alerts
* Severity Classification
* Docker Container Support
* GitHub Actions CI/CD Pipeline

## Technologies Used

* Python
* Flask
* SQLite
* Docker
* GitHub Actions

## Running Locally

```bash
pip install -r requirements.txt
python3 app.py
```

## Running with Docker

```bash
docker build -t cyberguard .
docker run -p 8082:8081 cyberguard
```
